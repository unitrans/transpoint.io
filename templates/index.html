About Transpoint

Transpoint is a Web-service which is application on top of the online translation services such as Google.Translator, Yandex.Translator, Bing.Translator and Abbyy Lingvo solutions

Benefits of using Transpoint is:

  • Reduced translations price (possible by using big quota with discount)
  • Possibility to save translations with specified identifier
  • Access saved translation is faster than request to Google.Translator
